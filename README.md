# Meu Projeto

Repositório criado para praticar contribuições com Pull Requests.

## Sobre

Este projeto é um ponto de partida para aprender o fluxo de trabalho com Git e GitHub.

## Como usar

1. Clone o repositório
2. Explore os arquivos
3. Abra uma issue ou PR com sua contribuição!

## Contribuindo

Contribuições são bem vindas. Siga os passos:

1. Faça um fork
2. Crie uma branch (`git checkout -b minha-melhoria`)
3. Faça commit das mudanças (`git commit -m "Adiciona minha melhoria"`)
4. Abra um Pull Request
